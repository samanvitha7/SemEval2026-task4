# DeBERTa Siamese Prototype

## Purpose
- Task/Track: Track A style pairwise ranking
- Goal: Single-script Siamese/contrastive DeBERTa experiment
- Status: Prototype

## Core File
- `Dberta_Siamese.py`: Notebook-exported training/evaluation script

## Usage
The script contains notebook/Colab commands and hardcoded paths.
Treat it as experimental reference code.

```bash
python DBERTA_SIAMESE/Dberta_Siamese.py
```

## Notes
- This is not the canonical DeBERTa pipeline.
- For maintained training/prediction with configs and checkpoints, prefer root `train.py` and `predict.py`.
